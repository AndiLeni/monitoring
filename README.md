# Monitoring Addon für Redaxo CMS

Das Addon stellt einen Monitoring-Service für Redaxo CMS bereit.
Damit lassen sich Webseiten hinsichtlich ihrer Funktionalität überwachen.
Cronjobs sollten verfügbar sein.
